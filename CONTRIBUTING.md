# Contributing to GenAI Reference Project

We welcome contributions from students, mentors, and external advisors.

## Guidelines
1. Fork the repository and create a feature branch
2. Submit pull requests (PR) to `main`
3. PR must include:
   - Description of the change
   - Any new dependencies
   - Basic tests if applicable
4. Avoid committing sensitive data (API keys, passwords)
5. Respect code style and folder structure
